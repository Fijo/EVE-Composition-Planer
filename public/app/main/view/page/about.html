Used Frameworks/ Tools

  • Javascript (Script language)
  • Underscore.js (Core framework)
  • JQuery (Core framework)
  • AngularJS (UI framework)
  • Angular-Cache (Caching framework for angular)
  • Karma (Test runner)
  • Jasmine (Javascript testing framework)
  • RequireJS (Javascript file and module loader)
  • npm (nodejs package manager)
  • Bower (package manager for web dependency ressources)
  • AngularStrap (Bootstrap UI elements for Angular, wana get rid of that replace it with Angular Bootstrap)
  • Angular Bootstrap (Bootstrap UI elements for Angular)
